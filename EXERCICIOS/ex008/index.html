Principais formações

Negrito

Para deixar um texto em negrito ou uma parte dele, uma frase ou parte dela basta usar a tag B

Outra tag q pode ser usada para deixar em negrito também é a STRONG que diferente da B possui semântica.

Italico ou Emfase

Para deixar algo em italico basta usar a tag I ou a tag EM que significa emfase e possui semantica

*Caso eu queira adicionar algo ja pronto dentro de uma tag, basta digitar a tag, selecionar oque desejo por nela e arrastar com o mouse (eu acho mais facil assim)

A tag MARK faz o serviço de uma caneta marca-texo!

Para trocar a cor do MARCADOR basta ir dentro da primeira parte da tag e dar um espaço apos o mark e digitar style e selecionar backgroung color e escolher a cor q desejar!

Porem caso eu use novamente o MARK ele não vai continuar com a alteração anterior de cor, para mudar a cor de todos os mark de uma só vez precisa criar uma TAG STYLE no cabeçalho entre o title e o head!

TAMANHOS DE TEXTO

Vamos ver tags para mudar o tamanhos dos textos, para grande ou pequeno!

A tag BIG está obsoleta, não se usa mais, mesmo que ainda funciona, serve para deixar a letra maior, porem isso se encaixa em forma e é feito com CSS

A tag SMALL ainda se usa, pois possui sentido e por isso não está obsoleta, em muitos casos de contratos por exemplo ela vem a ser util

DELETANDO/DESCARTANDO TEXTO

Para descartar uma parte do texto, dizer que ela não é mais valida, se usa a tag DEL!

SUBLINHANDO TEXTO

Para sublinhar textos ou palavras podemos usar a tag INS e tambem podemos usar a tag U porem esta não possui semantiga

TEXTO SOBRESCRITO

Para deixar algo sobrescrito como por exemplo x2 que seria x na raiz quadrada de 2 se usa a tag SUP

TEXTO SUBSCRITO

Para deixar textos subscritos como por exemplo H20 que seria o 2 da formula da agua se usa a tag SUB
