OUTRAS FORMATAÇÕES

Código-fonte

A tag CODE serve para criar um espaçamento entre as letras facilitando a leitura, usada para demonstrações de codigos ou sites conforme desejar.

A tag PRE usada antes 
                da CODE serve para ajustar 
                    as linhas da digitação dos codigos
                         de suas linguagens conforme o necessário.
    

Os comandos de SHIFT + TAB com todo o código selecionado
        permite ajustar ele mais para a esquerda conforme o 
        necessário da linguagem e só o TAB ajusta para a direita

Citações

A tag Q se usa para citaçõens mais simples, ela cria aspas " " na frase!

Citaçõens Completas

Para citaçõens mais completas se usa a tag BLOCKQUOTE que serve para quando vai se referir a um livro, um documento, caso possua o link do lugar que pegou a citação pode acrescentar dentro da tag para que o algoritimo saiba de onde você pegou a citação. ex: blockquote cite="URL"

Abreviaçõens

Para descrever o significado de abreviaçõens usamos a tag ABBR que mostra o significado quando passado o cursor em cima dela. ex DTR
